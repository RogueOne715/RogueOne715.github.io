    # 1.标题

    在markdown语法中使用文字前面使用#来表示标题，#的个数表示是几级标题，一共为六级。

    注意: #和文字之间需要空出一个空格。

    # 2.段落

    段落之间如果需要换行，要用enter键隔开一行，书写时如果只是换行，在页面上文字还是贴在一起的。

    示例：
    第一行文字。
    第二行文字。

    文字倾斜是在文字两侧使用*或者_

    # 5.列表

    # 5.1 无序列表

    无序列表是在文字前面加上- ，注意的是-和文字需要隔开一个空格。

    嵌套列表的写法是在需要嵌套的列表前面使用tab或者空格隔开四个制表符的距离

    示例：

    • 无序列表1
      • 嵌套无序列表1
      • 嵌套无序列表2
    • 无序列表2
    • 无序列表3
    • 无序列表4

    # 5.2 有序列表

    有序列表在文字前面加上数字.

    示例：

    1. 第二项
      1. 有序子项1
      2. 有序子项2
    2. 有序列表2
    3. 有序列表3
    4. 有序列表4

    # 6. 链接

    要添加链接需要如下语法：[链接文本](链接地址)

    链接的文本就是用户能看到的文字，地址就是用户点击会跳转的URL地址

    示例： 百度一下，你就知道 (opens new window)

    想到显示图片要如下语法：![图片的替代文字](图片的地址)
